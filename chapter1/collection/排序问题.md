两种方式实现：

* 集合中的元素类，这个类自己实现compareable接口
* 定义一个比较类，它实现了comparator接口

## Comparable

一个实现了`Comparable`接口的类，同一个类的两个对象之间要想比较，对应的类就要实现`Comparable`接口，并实现`compareTo()`方法

```java
import java.util.ArrayList;
import java.util.Collections;

class HDTV implements Comparable<HDTV> {
    public int size;
    public String brand;

    public HDTV(int size, String brand) {
        this.size = size;
        this.brand = brand;
    }

    @Override
    public int compareTo(HDTV tv) {

        if (this.size > tv.size)
            return 1;
        else if (this.size < tv.size)
            return -1;
        else
            return 0;
    }

    public static void main(String[] args) {
        ArrayList<HDTV> tvs = new ArrayList<HDTV>();
        HDTV tv1 = new HDTV(55, "Samsung");
        HDTV tv2 = new HDTV(60, "Sony");

        tvs.add(tv2);
        tvs.add(tv1);

        Collections.sort(tvs);

        for (HDTV i : tvs) {
            System.out.println(i.brand);
        }
    }
}
```

输出结果：

```
Samsung
Sony
```

## comparator

```java
import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;

class HDTV implements Comparable<HDTV> {
    public int size;
    public String brand;

    public HDTV(int size, String brand) {
        this.size = size;
        this.brand = brand;
    }

    @Override
    public int compareTo(HDTV tv) {

        if (this.size > tv.size)
            return 1;
        else if (this.size < tv.size)
            return -1;
        else
            return 0;
    }

    public static void main(String[] args) {
        ArrayList<HDTV> tvs = new ArrayList<HDTV>();
        HDTV tv1 = new HDTV(55, "Samsung");
        HDTV tv2 = new HDTV(60, "Sony");

        tvs.add(tv2);
        tvs.add(tv1);

        Comparator<HDTV> comparator = new Comparator<HDTV>() {

            @Override
            public int compare(HDTV o1, HDTV o2) {
                return o1.size - o2.size;
            }

        };

        Collections.sort(tvs, comparator);

        for (HDTV i : tvs) {
            System.out.println(i.brand);
        }
    }
}
```

输出结果：

```
Samsung
Sony
```
